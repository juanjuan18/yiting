城市路内停车收费系统

易停2016-10-19

固定咪表收费方案

概述

易停人工手持咪表收费管理系统，由手持咪表、巡检终端、地磁探测器、管理器、云平台等构成。手持咪表支持刷卡收费，结合艾科智泊停车管理云平台，可实现信息实时交互，准确记录数据，可有效解决逃单、乱收费等问题，并为欠逃费车辆追缴提供依据，有效的提高停车管理监控力度和停车服务信息化水平。

架构图
特点
  • 1、车主自主登记、缴费，无需收费员，减少人力，降低了运营成本；
  • 2、解决了人工收费长期存在的收费漏洞问题；
  • 3、为诱导信息发布提供了实时数据支撑；
  • 4、为政府/运营单位提供了车位占用情况统计分析，为决策提供了数据支撑；
  • 5、开放的数据接口，系统扩展能力强。
案例-佛山南海桂城路边停车项目

佛山市南海区桂城路内停车智能收费系统一期建设工程（东小片区，简称桂城一期项目）桂澜路以东、佛山市一环路以西、佛平路以南、怡海路以北的区域内，涉及城市花园西门间隔路、新天地北门间隔路、鹿景村南门间隔路、南海妇幼西侧间隔路、东平路、华翠南路、简平路、实验幼儿园北面间隔路、平海路、庆怡路、庆平路、庆安路等12个路段，共995个停车位。
